Algoritm pentru transformarea din baza 10 in baza b

 
p = 1;
x = 0;

while (nr != 0) // cat timp numarul nu este 0
{
    c = nr % b; // restul impartirii nr la b
    x = x + p * c; // construim numarul in baza b
    p = p * 10;
    nr = nr / b;
}
        // x contine nr in baza b

Observatii:

  1. Cifrele numerelor scrise in baza b (b<=10) sunt intre 0 si b-1.
  2. Pentru a transforma din baza b in baza c se face trecerea prin baza 10 (b => 10 => c)
